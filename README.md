# TripNest 🌍

TripNest is a full-stack travel accommodation platform inspired by Airbnb. It enables users to discover destinations, explore unique stays, create property listings, upload images, and share reviews through an intuitive and responsive interface.

## ✨ Features

* User Authentication & Authorization
* Create, Edit, and Delete Listings
* Image Uploads with Cloudinary
* Reviews and Ratings System
* Interactive Maps with Leaflet
* Category-Based Listing Filters
* Responsive Design for All Devices
* Flash Messages and Error Handling

## 🛠️ Tech Stack

**Frontend:** EJS, HTML, CSS, Bootstrap, JavaScript

**Backend:** Node.js, Express.js

**Database:** MongoDB Atlas, Mongoose

**Authentication:** Passport.js

**Cloud Storage:** Cloudinary
